# Subeom Kwon Portfolio — Modern Static Version

This is a modernized static portfolio that replaces the old school-project-heavy version with a current professional profile focused on AI/ML infrastructure, Kubernetes, GPU inference operations, backend systems, and data migration work.

## Files

- `index.html` — main page
- `styles.css` — responsive styling
- `main.js` — mobile navigation, reveal animation, active section highlighting
- `Subeom_Kwon_Resume.pdf` — resume linked from the page

## Deploy

You can deploy the folder as-is to GitHub Pages, Netlify, Vercel static hosting, S3 + CloudFront, or any Nginx static directory.
